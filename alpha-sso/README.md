# oauth2原理
```
OAuth（开放授权）是一个开放标准，
允许用户授权第三方移动应用访问他们存储在另外的服务提供者上的信息，
而不需要将用户名和密码提供给第三方移动应用或分享他们数据的所有内容，
OAuth2.0是OAuth协议的延续版本，但不向后兼容OAuth 1.0即完全废止了OAuth1.0。

```
